## Design Philosophy
This ecosystem focuses on building a foundation of powerful and independent **Skills**. Subsequently, **Agents** are defined to utilize these skills through specific **Workflows**.

## Directory Structure
- **/skills**: Contains reusable skill modules and tools.
- **/agents**: Defines roles, instructions, and configurations for each Agent.
- **/workflows**: Operational processes and action sequences for the Agents.
- **/prompts**: A repository for prompt templates (system, task templates).
- **/Infrastructure**: Architectural standards, common utilities, and technical documentation.
- **/tasks**: Project progress management and system task lists.
